室管膜瘤

1. 概念

儿童室管膜瘤（Ependymoma）是一种起源于脑室或脊髓中央管的室管膜细胞的肿瘤，属于神经上皮性肿瘤。室管膜瘤占所有儿童中枢神经系统肿瘤的5%~10%，是儿童第三常见的脑肿瘤（仅次于髓母细胞瘤和胶质瘤）。它可发生于任何年龄，但多见于3岁以下儿童，男性发病率略高于女性。

室管膜瘤好发于第四脑室、侧脑室和脊髓，具有局部侵袭性，易复发，但远处转移较少见。

2. 疾病分类

根据世界卫生组织（WHO）分类，室管膜瘤可分为以下类型：

（1）按恶性程度分级

  • Ⅰ级（低级别）：
    • 室管膜下瘤（Subependymoma）：生长缓慢，预后良好。
    • 黏液乳头型室管膜瘤（Myxopapillary Ependymoma）：多见于脊髓圆锥和马尾区，预后较好。
  • Ⅱ级（低级别）：
    • 经典室管膜瘤（Classic Ependymoma）：最常见，生长较慢，但易复发。
  • Ⅲ级（高级别）：
    • 间变性室管膜瘤（Anaplastic Ependymoma）：恶性程度高，生长快，预后较差。

（2）按部位分类

  • 幕上室管膜瘤：位于大脑半球，占30%~40%。
  • 幕下室管膜瘤：位于第四脑室或小脑，占60%~70%。
  • 脊髓室管膜瘤：多见于脊髓中央管，占10%~20%。

3. 病因

室管膜瘤的具体病因尚不明确，但可能与以下因素有关：

（1）遗传因素

  • 基因突变：如NF2、LAMA2、RELN等基因突变与室管膜瘤的发生密切相关。
  • 遗传综合征：某些遗传性疾病患儿室管膜瘤发病率较高，例如神经纤维瘤病2型（NF2）和Li-Fraumeni综合征。

（2）环境因素

  • 电离辐射：暴露于高剂量电离辐射可能增加室管膜瘤风险。
  • 化学物质：某些致癌物质（如亚硝酸盐）可能与肿瘤发生有关。

（3）其他因素

  • 病毒感染：某些病毒（如巨细胞病毒）可能与肿瘤发生有关。
  • 免疫因素：免疫系统功能异常可能参与肿瘤发生。

4. 临床症状

室管膜瘤的症状与肿瘤部位、大小及脑脊液循环障碍有关，常见症状包括：

（1）颅内压增高

  • 头痛：早晨加重，可能伴有恶心、呕吐。
  • 视乳头水肿：视力下降或视野缺损。
  • 意识改变：嗜睡、烦躁或昏迷。

（2）部位特异性症状

  • 幕下室管膜瘤：
    • 共济失调：行走不稳、动作不协调。
    • 脑神经麻痹：面瘫、吞咽困难、复视。
  • 幕上室管膜瘤：
    • 癫痫发作：部分性或全身性癫痫。
    • 运动障碍：肢体无力、偏瘫。
  • 脊髓室管膜瘤：
    • 背痛：局部疼痛或放射性疼痛。
    • 运动障碍：肢体无力、行走困难。
    • 感觉异常：麻木、刺痛。

（3）其他症状

  • 内分泌异常：如尿崩症、生长发育迟缓。

5. 诊断依据

室管膜瘤的诊断需结合临床表现、影像学检查和病理学检查：

（1）影像学检查

  • 颅脑MRI：首选检查方法，可清晰显示肿瘤位置、大小及与周围组织的关系。
  • 全脑全脊髓MRI：评估肿瘤是否播散至脊髓。

（2）病理学检查

  • 活检或手术切除：获取肿瘤组织进行病理学诊断，明确肿瘤类型和分级。
  • 分子病理学检测：检测NF2、LAMA2等基因突变，指导治疗和预后评估。

（3）脑脊液检查

  • 细胞学检查：评估肿瘤是否播散至脑脊液。

6. 治疗方法

室管膜瘤的治疗需根据肿瘤类型、部位和患儿年龄制定个体化方案，主要包括：

（1）手术治疗

  • 完全切除：尽可能完全切除肿瘤，缓解颅内压增高症状。
  • 部分切除：适用于肿瘤与重要结构粘连紧密的情况。

（2）放射治疗

  • 局部放疗：适用于术后残留或高级别室管膜瘤。
  • 全脑全脊髓放疗：适用于肿瘤播散至脊髓的患儿。

（3）化学治疗

  • 适应症：婴幼儿（避免放疗）或复发室管膜瘤。
  • 常用药物：顺铂、环磷酰胺、依托泊苷等。

（4）靶向治疗

  • 适应症：复发或难治性室管膜瘤。
  • 常用药物：EGFR抑制剂（如厄洛替尼）、VEGF抑制剂（如贝伐单抗）。

（5）免疫治疗

  • 临床试验：包括免疫检查点抑制剂、CAR-T细胞疗法等。

7. 预后

室管膜瘤的预后因肿瘤类型、部位和治疗方式而异：

（1）预后因素

  • 肿瘤分级：低级别室管膜瘤预后优于高级别室管膜瘤。
  • 手术切除程度：完全切除可显著改善预后。
  • 肿瘤部位：幕上室管膜瘤预后优于幕下室管膜瘤。
  • 年龄：3岁以上患儿预后优于3岁以下患儿。

（2）生存率

  • 5年无事件生存率（EFS）：
    • 低级别室管膜瘤：70%~80%。
    • 高级别室管膜瘤：30%~50%。
  • 5年总生存率（OS）：
    • 低级别室管膜瘤：80%~90%。
    • 高级别室管膜瘤：50%~70%。

（3）后遗症

  • 神经功能障碍：共济失调、智力低下、学习困难。
  • 内分泌异常：生长激素缺乏、甲状腺功能减退。
  • 继发肿瘤：放疗后可能继发脑膜瘤、胶质瘤等。
